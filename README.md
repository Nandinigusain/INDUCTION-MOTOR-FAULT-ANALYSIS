# Induction Motor Analysis using Simulink

## Description
This project models and analyzes an induction motor using MATLAB Simulink.
The model includes electrical and mechanical subsystems and is used to
study motor behavior under different conditions.

## Software Required
- MATLAB
- Simulink
- Simscape Electrical

## How to Run
1. Open MATLAB
2. Open `Induction_Analysis.slx`
3. Click Run
4. Observe outputs on scopes
